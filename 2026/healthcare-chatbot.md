# Healthcare Chatbot using AI

## Name
Sivagami

## Project Title
Healthcare Chatbot using Natural Language Processing

## Description
This project aims to develop an AI-powered healthcare chatbot that can provide basic medical guidance and answer health-related questions. The chatbot will use Natural Language Processing (NLP) to understand user queries and respond with relevant information.

The system can help users get quick health advice, understand symptoms, and provide suggestions on when to consult a doctor.

## Tech Stack
Python
Natural Language Processing
Machine Learning
TensorFlow / NLP libraries
